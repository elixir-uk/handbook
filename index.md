A living reference for who ELIXIR-UK is, how we sound, who we talk to, and how the community shows up alongside us.

This site is the single source of truth for four things:

- **[Brand](/brand)** — the visual identity. Logo, colour, typography, imagery, templates.
- **[Style guide](/style-guide)** — how we write. Voice, lexicon, do-and-don't, and the LLM system prompt that lets a language model draft on-brand copy on our behalf.
- **[Communications strategy](/communications-strategy)** — who we're talking to, why, on what channels, and how we know it's working.
- **[Community participation model](/community-participation-model)** — how researchers, trainers, and partners engage with ELIXIR-UK and grow into deeper roles over time.

## How this site is built

Every page is a markdown file in a public GitHub repo. That means:

- the content is version-controlled — every change is reviewable
- the same markdown that renders this site can be fed to a language model as a system prompt, so the model writes in our voice without us maintaining a parallel "prompt document"
- contributions go through pull requests
